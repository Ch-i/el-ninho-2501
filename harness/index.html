cycle pipeline · loop_once()

Evidence Cycle

Each cycle runs nine deterministic stages. Evidence gates at 6b prevent a failed command from being logged as a finding. A 3-strike blocker triggers voice warnings and automatic avoidance injection.

01Snapshotengagement.snapshot() reads current phase, findings count, and blocker history from Redis. Seeds the context window.redis
02Target Mapprocess_memory.attach_to_map() loads running services, known credentials, and paths discovered in previous cycles.state
03Hypothesesreason_hypotheses() generates 2–4 attack hypotheses. Dolphin feeds on findings + recent cycles + writeup context to reason forward.dolphin
04Researchplan_research() + run_research() queries DuckDuckGo and reads pages for CVE details, service versions, and exploit patterns.web
05Decisionchoose_decision() selects the next action + command. Injects all previously failed commands and repeated blockers to force avoidance.hermes
06Executionexecute_decision() runs shell_exec / web_search / context_exec. Returns raw artifact + duration. All commands allowed — no blocklist.shell
6bClosure Gateevidence_gates.assess_cycle() deterministically checks artifact patterns. Must match expected evidence before the cycle logs a finding.gate
6cRecoveryrecord_blockers() fingerprints failures. 2× repeat → voice warning + avoidance injection. 3× → TERMINAL_BLOCK → mission paused.safety
07–08Map + Recordupdate_map() extracts services, paths, creds. engagement.log_cycle() + vault.log_full_cycle() persist to Redis and Obsidian vault.vault
tool surface · tools.py

Four Tools

Agents are restricted to four primitives. Everything the harness can do flows through one of these — making the execution envelope explicit and auditable.

⚡
shell_exec
Runs any shell command locally. No allowlist. 60s timeout. Full stdout/stderr. Used by Hermes for all target interaction.
🌐
web_search
DuckDuckGo HTML scrape. 5s rate limit. Used for CVE research, exploit PoC hunting, and service fingerprint lookup.
📄
web_read
Fetch + text-extract a URL. Strips HTML, returns clean prose. Reads exploit docs, GitHub source, NVD advisories.
🔗
context_exec
SSH / container / K8s context routing. Runs commands on remote targets via registered execution contexts.
